## 微信推送

文本消息
```yaml
service: notify.home_cloud_wechat
data:
  message: 微信推送消息
```
图文消息
```yaml
service: notify.home_cloud_wechat
data:
  message: 微信推送消息
  data:
    image: https://www.home-assistant.io/images/favicon-192x192.png
```
